<pre>
* To install Tkinter: *

Ubuntu:
sudo apt-get install python-tk
sudo apt-get install python3-tk

OS X (requires Homebrew):
xcode-select --install
brew install tcl-tk
brew install python --with-tcl-tk

Windows:
(Included in standard Python distribution; run the IDLE editor.)


* To import Tkinter: *
Python 2:
import Tkinter

Python 3:
import tkinter
