<pre>
* To install Tkinter: *

Ubuntu:
sudo apt-get install python-tk
sudo apt-get install python3-tk

OS X:
Requires Homebrew:
https://docs.python-guide.org/starting/install3/osx/

Windows:
Included in python.org Python distribution; run the IDLE editor.


* To import Tkinter: *
Python 2:
import Tkinter

Python 3:
import tkinter
